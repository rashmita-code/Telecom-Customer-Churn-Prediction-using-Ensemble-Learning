\# 📊 Telecom Customer Churn Prediction using Ensemble Learning



\## 📌 Overview

Customer churn prediction is a critical problem in business analytics. This project builds an advanced machine learning pipeline to predict whether a customer is likely to leave a service based on behavioral and usage data.



The project not only focuses on prediction but also on extracting meaningful insights to support data-driven decision-making.



\---



\## 🎯 Objectives

\- Perform comprehensive data preprocessing  

\- Conduct in-depth exploratory data analysis (EDA)  

\- Identify key features influencing churn  

\- Build and compare multiple machine learning models  

\- Evaluate and optimize model performance  



\---



\## 📂 Dataset

\- `churn-bigml-80.csv` → Training dataset  

\- `churn-bigml-20.csv` → Testing dataset  



\### 🔑 Key Features:

\- Account Length  

\- International Plan  

\- Voice Mail Plan  

\- Total Day/Evening/Night Minutes \& Charges  

\- Customer Service Calls  

\- Target Variable: \*\*Churn (True/False)\*\*

\### 1. Data Preprocessing
